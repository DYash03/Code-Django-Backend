Welcome to the Editor section

Now everyone can be an editor

Given below are the guidelines required to be an editor :

  1. Enter your full name.

  2. Enter the name or the title of your program name.

  3. Mention your program source. If it is copied from Hakerank , CodeChef etc then mention it. If source is not known for any reason write unknow.

  4. Enter your email next.

  5. Enter the description of your program.

  6. Next provide your solution.

    Note: It is not necessary that same solution will be posted. But if the solution is wrong your code will not be uploaded.

  7. Finally give atleast 1 sample input and sample out respectively.
